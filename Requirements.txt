Network Scanner Project — Requirements

This project uses only the Python Standard Library.
No external packages are required or used.

Standard Library modules used:
   - socket          (TCP connections)
   - ipaddress       (CIDR/subnet parsing)
   - concurrent.futures (multi-threaded scanning)
   - argparse        (CLI interface)
   - sys             (exit handling)
   - time            (timing/progress)
   - datetime        (report timestamps)
   - collections     (defaultdict for report grouping)

Python version requirement: Python 3.10+
